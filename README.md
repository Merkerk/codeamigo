# codeamigo

### Installation

```
cd server
yarn

cd client
yarn
```

### Dev/Server

```
cd server
yarn watch // ts => js
yarn dev // run index.js
brew services start postgresql // start our db
sudo redis-server /usr/local/etc/redis.conf // start redis
```

Visit http://localhost:4000/graphql

### Dev/Client

```
cd client
yarn dev
```

Visit http://localhost:3000

### Stack/Requirements

```
1. postgresql
2. redis
3. express
4. apollo-server
5. next.js
6. tailwindcss
```
